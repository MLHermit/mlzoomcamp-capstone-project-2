# mlzoomcamp-capstone-project-2
mlzoomcamp capstone project 2 repo
The development and deployment of cnn model with pytorch for image classification which automatically categorize new product listings. It will assist in inventory management by quickly sorting items. It categorizes images of clothing items into distinct garment types such as shirts, trousers, shoes, etc.
